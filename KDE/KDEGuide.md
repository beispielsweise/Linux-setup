##### Theme used: WhiteSur Dark 
Install KvantumManager, go onto https://store.kde.org/p/1398841/ and install the theme inside of KvantumManager. 
In settings -> Application style choose Kvantum-dark for blur to work. 

##### For Desktop Effects install: 
https://github.com/ekaaty/kwin-effect-rounded-corners 
Set Outline for active window to get hyprland type window borders (note: refreshes only on cursor movement or window updates via keyboard, bugged)
Increase corner Roundness to ~12, to avoid visual artefacts

##### For KwinScripts install:
Khronkite
Setup Geometry, under Options remove ticks for individual VD, enable Remove Titlebar

##### ZSH
1. Install ZSH
2. Copy .zshrc file, download plugins listed under source, drop into a necessary directory
3. to setup powerlevel10k, remove [[ ... ]] line in the config, fun through configuration, repaste it somewhere in the file.
